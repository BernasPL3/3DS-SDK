# 3DS-SDK

A simple and open-source Software Development Kit for creating Nintendo 3DS homebrew applications and games.

## Features

* Easy project creation
* Command-line interface
* Built-in project templates
* Example projects
* Homebrew-focused workflow
* Cross-platform development support
* Lightweight and beginner-friendly

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/3DS-SDK.git
cd 3DS-SDK
```

## Quick Start

Create a new project:

```bash
3ds-sdk new MyGame
```

Build the project:

```bash
3ds-sdk build
```

Run diagnostics:

```bash
3ds-sdk doctor
```

## Available Commands

```text
3ds-sdk new <project>
3ds-sdk build
3ds-sdk clean
3ds-sdk run
3ds-sdk package
3ds-sdk install
3ds-sdk doctor
3ds-sdk version
3ds-sdk examples
3ds-sdk help
```
